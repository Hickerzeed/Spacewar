# Steam Explorer

这是一个简单的Steam游戏启动工具，帮助用户快速浏览并启动Steam游戏。

## 功能

- 提供图形界面浏览文件系统
- 支持选择并启动Steam游戏可执行文件
- 简单易用的界面

## 使用方法

1. 运行`steamexplorer.py`脚本
2. 在文件浏览器中导航到Steam游戏目录（通常在`D:\Steam\steamapps\common`）
3. 选择游戏的可执行文件(.exe)启动游戏

## 要求

- Python 3.x
- tkinter库（Python标准库）
